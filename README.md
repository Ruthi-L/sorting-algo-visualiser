# Sorting Algorithm Visualizer
A Flask-based implementation to visualize the step-by-step execution of sorting algorithms.

## Key Features
* **Asynchronous Rendering:** Uses a RESTful API to send array states from Python to JavaScript.
* **Complexity Insight:** Demonstrates the $O(n^2)$ behavior of Bubble Sort visually.
* **Architecture:** Modular separation of backend logic (Python) and frontend rendering (HTML/JS).

## How to Run
1. Install dependencies: `pip install -r requirements.txt`
2. Run the app: `python app.py`
3. Visit `http://127.0.0.1:5000` in your browser.


The aim here is to visualize sorting algorithms so as to make learning and understanding execution of various algorithms and their time complexities easier for students.
